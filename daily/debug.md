# ZEN of DEBUG

1. **不要试图消灭，我们只能共生。**
2. **不要放弃捕捉，有梦想是好事。**
3. **不要怀疑机器，它们从不出错。**
4. **不要相信自己，无论有多简单。**
5. **不要闭环思维，肯定你没想到。**
6. **不要妄图奇迹，那还不如玄学。**
7. **我很耐心，我很耐心？我很耐心。。。**
8. **我是瞎子？我是智障。我是傻逼！**
9. **不要和自己过不去，我的头发还有很多可以掉。**
