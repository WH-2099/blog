### 通用快捷键

| 基础键 | 指令键 | 命令 |
| ---: | :---: | :--- |
| **`CTRL`** | **`S`** | **保存** |
| **`CTRL`** | **`C`** | **复制** |
| **`CTRL`** | **`V`** | **粘贴** |
| **`CTRL`** | **`F`** | **查找** |
| **`CTRL`** | **`P`** | **打印** |
| **`ALT`** | **`F4`** | **关闭** |
